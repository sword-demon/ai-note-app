## 欢迎来到 AI Note

欢迎来到 AI Note 博客!这是我们的第一篇文章,标志着这个项目的正式启动。

### 什么是 AI Note?

AI Note 是一个基于 **Next.js** 和 **Fumadocs** 构建的现代化笔记和文档管理应用。我们致力于为开发者和内容创作者提供最佳的写作和阅读体验。

### 主要特性

- **MDX 支持**: 使用 MDX 编写内容,结合 Markdown 的简洁和 React 组件的强大
- **智能搜索**: 全文搜索功能,快速找到你需要的内容
- **深色模式**: 完美的深色和浅色主题支持
- **响应式设计**: 在任何设备上都有出色的体验

### 技术栈

我们使用了以下技术:

```typescript
const techStack = {
  framework: "Next.js 16",
  content: "Fumadocs MDX",
  styling: "Tailwind CSS 4",
  deployment: "Vercel",
};
```
